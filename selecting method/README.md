# 台股資金流向 & 類股輪動 Dashboard

以 Streamlit 建置的台灣股市資金流向分析儀表板，整合 TWSE OpenAPI、三大法人資料與 yfinance 歷史價格，提供即時類股輪動視覺化。

## 功能

- **總覽**：前五大類股 KPI、成交比重長條圖、今日異常警示摘要
- **產業熱圖**：Treemap，以成交比重為大小、漲跌幅為顏色
- **資金聚集泡泡圖**：X 軸（成交比重變化）× Y 軸（5 日漲跌幅）× 大小（成交值）
- **領頭羊**：資金前五大類股，各取漲幅前三名個股及迷你走勢圖
- **警示面板**：成交量暴增、三大法人資金流入、概念股領漲警示（高／中／低優先）
- **原始資料**：所有資料集可預覽並下載為 CSV

## 安裝

```bash
python -m pip install -r requirements.txt
```

## 啟動

```bash
python -m streamlit run app.py
```

啟動後瀏覽器會自動開啟，預設網址為 `http://localhost:8501`。

## 側邊欄設定

| 設定 | 說明 |
|------|------|
| 顯示概念股分群 | 切換是否在熱圖與流向中顯示自訂概念股群組 |
| 類股篩選 | 多選，留空代表顯示全部類股 |
| 階段篩選 | 噴發／築底／震盪／衰退，留空代表全部 |

## 資料來源

| 資料 | 來源 | 快取時間 |
|------|------|----------|
| 類股指數 / 個股成交 / 大盤成交值 | TWSE OpenAPI | 1 小時 |
| 三大法人買賣超 | TWSE RWD JSON（備援：HTML 爬蟲） | 1 小時 |
| 個股歷史價格 / 加權指數 | yfinance | 24 小時 |

資料來源狀態（✅ / ⚠️ / ❌）顯示於側邊欄底部。

## 概念股設定

預設追蹤以下概念股分群，可於 `config.py` 的 `CONCEPT_GROUPS` 自行新增或修改：

- **光通訊**：1491、4904、6285、3308、3491
- **低軌衛星**：3813、6579、4977、6269
- **記憶體**：3008、2408、2303
- **半導體設備**：3450、6533、5285

## 類股階段定義

| 階段 | 條件 |
|------|------|
| 噴發 | 股價 > MA50 > MA200，且 5 日內穿越 MA50 |
| 築底 | 股價 > MA200，股價 < MA50，MA50 斜率正向 |
| 震盪 | 股價距 MA50 在 ±3% 內，均線斜率近平 |
| 衰退 | 股價 < MA50 < MA200 |

## 檔案結構

```
app.py          # Streamlit 主程式，UI 與 6 個分頁
config.py       # 端點、類股對照、概念股名單、分析閾值
data_fetcher.py # 資料擷取層（TWSE API → 爬蟲 → yfinance）
analyzer.py     # 分析層（Money Flow、動能、階段判斷、異常偵測）
requirements.txt
```
